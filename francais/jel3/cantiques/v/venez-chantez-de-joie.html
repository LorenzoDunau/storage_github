Venez, chantez de joie

J'aime l'Eternel n°836

1. Venez, chantez de joie pour le Seigneur

Et criez vers Lui, le Rocher de notre salut ;

Le cœur reconnaissant allons vers Lui,

Exaltons le Seigneur par nos chants.

Chœur :

Allélu, alléluia, chante : alléluia ;

Allélu, alléluia, chante : alléluia ;

2. Prosternons-nous pour l'adorer,

A genoux, célébrons notre Créateur ;

Il est notre Dieu, nous Lui appartenons,

Nous sommes Son peuple, Il est notre Berger.

Chœur :

Allélu, alléluia, chante : alléluia ;

Allélu, alléluia, chante : alléluia ;
