J'aime l'Eternel n°672

1. Je confesse

Devant les hommes, devant les anges :

"Jésus-Christ est mon Seigneur."

Je proclame

Devant tous les peuples :

"Jésus est mort pour moi,

Je suis racheté."

2. Aujourd'hui, Deu agit avec puissance.

Il répand son Esprit sur ses enfants.

Dieu transforme nos nations par l'intercession.

Il répond à la foi de ses enfants.

3. Nous confessons

Devant les hommes, devant les anges :

"Jésus-Christ est le Seigneur."

Proclamons

Devant tous les peuples :

"Jésus est mort pour nous,

Nous sommes rachetés."
